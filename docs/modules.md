
# Module Documentation

## app.py
Main Flask application file that serves the dashboard and handles API requests.

## api/ai_client.py
Handles communication with external AI APIs and formats responses.

## api/system_stats.py
Collects and formats system performance statistics.

## api/miner_stats.py
Collects and formats mining statistics from XMRig.

## templates/dashboard.html
Main dashboard template with chat interface and widgets.

## static/style.css
Styling for the dashboard interface.

## static/app.js
Frontend JavaScript for interactivity and WebSocket communication.

## config/settings.env
Environment configuration file.

## services/startup.sh
Startup script for initializing services.

## services/rpi-dashboard.service
Systemd service configuration.
